## Environment

All code in this repository is written and executed in the **[marimo](https://marimo.io/)** notebook environment.  

- To run the code, you **must use marimo**. You can install it via pip:

  ```bash
  pip install marimo
  ```

  Then run a notebook with:

  ```bash
  marimo run notebook.py
  ```

  or edit interactively with:

  ```bash
  marimo edit notebook.py
  ```

- For convenience, **static HTML exports** of each notebook are also provided. You can view these directly in your browser without installing marimo.

## Contents

Each assignment is placed in its own folder (e.g., `HW/HW_1`) and includes:
- A `.py` marimo notebook file
- A corresponding `.html` static view
- Any necessary data or supplementary files (if applicable)
